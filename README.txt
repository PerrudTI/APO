# Gerador de Currículo

Este projeto é um gerador de currículos desenvolvido utilizando PHP, HTML, CSS (Tailwind CSS) e JavaScript. Ele permite que os usuários insiram suas informações pessoais, experiências profissionais e formação acadêmica, e gere um currículo em formato PDF. Além disso, oferece a funcionalidade de compartilhar o currículo nas redes sociais.

## Funcionalidades

- Inserir dados pessoais (nome, data de nascimento, endereço, contato)
- Adicionar múltiplas experiências profissionais
- Adicionar múltiplas formações acadêmicas
- Gerar currículo em PDF
- Compartilhar currículo no Facebook, Twitter e LinkedIn

## Tecnologias Utilizadas

- PHP
- HTML
- CSS (Tailwind CSS)
- JavaScript
- jsPDF (para gerar PDFs)
- Font Awesome (para ícones de redes sociais)
